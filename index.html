About Me

I'm Star Ayan, a FrontEnd Developer who turns interface ideas into fast, accessible, production-ready code. My core toolkit is HTML5, CSS3, Tailwind CSS v4, modern JavaScript (ES6+), TypeScript, and React.js — the stack I use to build interfaces that feel considered rather than assembled. I care about the details that separate a good site from a great one: semantic markup, clean component architecture, and layouts that hold up on any screen.

Day to day, I build fully responsive websites and modern UI/UX, assemble reusable component libraries, and tune performance so pages load fast and stay smooth. I've shipped landing pages, personal and business portfolios, and full e-commerce storefronts, and I regularly convert Figma designs into pixel-accurate, responsive websites. Every line I write aims to be clean, maintainable, and easy for the next developer to pick up.
